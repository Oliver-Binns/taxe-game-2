Setup Problems

Eclipse says "Android Development Tools will be ignored because it is already installed, and updates are not permitted." when trying to install the Android Plugin.

Eclipse says that the location of the Android SDK has not been setup in the preferences

  1. Open Eclipse preferences (Window -> Preferences)
  2. Select Android
  3. Browse for the SDK folder. Click Apply.
  4. Click OK.

I get the Error "Error in runnable 'Creating Gradle model' when trying to Import the project.

Eclipse 4.4.1 does not work with the Gradle Eclipse plugin 3.6.). The solution is to edit the eclipse.ini file for your install.

  1. Close Eclipse
  2. Open your installation location for Eclipse
  3. Add the following lines to the ini:
    • Dosgi.configuration.area.default=null
    • Dosgi.user.area.default=null
    • Dosgi.user.area=@user.dir
  4. Restart Eclipse

This should fix the problem. However, if you encountered this problem when trying to use the Computer Science computers and their version of Eclipse you will have to install a fresh copy of the latest version of Eclipse along with the relevant plugins.
